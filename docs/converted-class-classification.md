# Classificação das classes convertidas

Os 31 arquivos estavam reunidos em `Classes/Prestige/`, mas o texto de
**Class Information** dos próprios arquivos os separa em dois grupos:

- 13 Advanced Classes;
- 18 Prestige Classes.

Na conversão para PF1e, essa classificação foi preservada:

- Advanced Class: `system.subType: "base"`;
- Prestige Class: `system.subType: "prestige"`.

## Observação

O diretório onde os Markdown estavam armazenados não foi usado para determinar
o tipo da classe. A classificação acima foi extraída da declaração
`advanced class` ou `prestige class` presente no conteúdo de cada arquivo.
